# Platonus Test Trainer

Сайт для студентов для практики тестов Платонус. Работает полностью в браузере и сохраняет историю локально.

## Возможности

* Импорт тестов (`<question>` / `<variant>`, первый вариант — правильный).
* Список загруженных тестов.
* **Cram Mode** — деление на мини-тесты.
* **Test Mode** — 30 случайных вопросов.
* Настройки, история, адаптивный дизайн.

## Как пользоваться

1. Открой сайт: **[https://azadron228.github.io/](https://azadron228.github.io/)**
2. Загрузите файл с тестом.
3. Выберите тест из списка и начните обучение.
